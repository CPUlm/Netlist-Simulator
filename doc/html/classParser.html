Detailed Description

The syntax analyser for the Netlist language.

The parser takes a Lexer as input and returns a parsed Netlist Program. If any error is found, a null program is returned. Furthermore, the parser stops at the first error encountered. All errors and warnings are emitted using the report API via the given ReportManager instance.

If you are curious, the parser is implemented internally using the recursive descent algorithm.
